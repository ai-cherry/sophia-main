### Installation

1. **Clone the repository**
   ```bash

   cd sophia-main
   ```

2. **Set up environment**
   ```bash
   cp .env.example .env
   # Configure with your API keys and credentials
   ```

3. **Install dependencies**
   ```bash
   make install
   ```

4. **Start the natural language control system**
   ```bash
   make dev
   ```
